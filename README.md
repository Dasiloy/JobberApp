 
# JobberApp

## Description

JobberApp is a simple job board application that allows users to apply for jobs. Users can also view job listings and apply for jobs.

## Features

- Users can view job listings
- Users can apply for jobs
- Users can view their job applications


## Technologies

- React
- Node.js
- Express
- MongoDB
- React Native

## Installation

1. Clone the repository
2. Install dependencies
3. Run the server
4. Run the client
   
   ```bash
   git clone (https://)
   cd JobberApp
   npm install
   npm start
   ```

## Usage

1. Register as a user
2. View job listings
3. Apply for jobs
4. View your job applications
